**直接使用：**

```go
package main

import "fmt"

func main() {
	arr := []string{"1", "2"}

	for _, v := range arr {
		fmt.Println(v)
	}
}
```
{{< block type="tip" title="输出">}}
```text
1
2
```
{{< /block >}}

**传递函数**

```go
package main

import (
	"fmt"
	"time"
)

func main() {
	arr := []string{"1", "2"}

	for _, v := range arr {
		go func() {
			fmt.Println(v)
		}()
	}

	time.Sleep(time.Second)
}
```

{{< block type="tip" title="输出">}}
```text
2
2
```
{{< /block >}}
